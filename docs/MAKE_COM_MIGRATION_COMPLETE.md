## ⚠️ Important: DO's and DON'Ts

### ✅ DO
- Commit code to git (make_cloud_pipeline.py, etc.)
- Commit config templates (make_env_template.json)
- Test locally before deploying

- Commit lambda_deployment/ directory
- Hardcode credentials in code
- Run pipeline from 2 places simultaneously
- Share webhook URLs publicly
- Delete Lambda without backup
